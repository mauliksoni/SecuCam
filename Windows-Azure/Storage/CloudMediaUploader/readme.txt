This is all the instructions to have this program installed and running.

1. This program have a settings.config file and everything is based on that file.
2. It uploads video files from any folder as specified by "folderPath" setting in setting.config file
3. After uploading to the azure cloud, it deletes the file from the folder
4. please install pip https://pip.pypa.io/en/latest/installing.html
5. please install python sdk for windows azure ( pip install azure ) or https://github.com/Azure/azure-sdk-for-python
6. please run start.py (python start.py)
